# 情報工学科とは
####コンピュータの基礎から、高度なプログラミングまで。情報化社会をさらに進化させる、IT業界の新たなリーダーを育成。
高度情報化が進展している現代社会において、ITを利活用できる*知識*や*技能*は**必要不可欠**です。

２年次からコースを選択し、専門的な学びを深めます。
1. システムエンジニアリングコース
2. インターネット＆セキュリティコース
3. 知能情報コース
